# RPG Supporting App
#
# Este projeto é para dar suporte para as pessoas que estão jogando 
# jogos de RPG.
# O suporte conta com:
# 1. Sorteio de dado
# 2. Nome e quantidade de participantes
# 3. Itens (ou cartas) que cada participante adquire ou desfaz no jogo
# 4. Level do personagem
#
# Além disso, a aplicação se propõe a: 
# 1. Deixar salvo as informações 
# 2. Resetar o jogo, caso solicitado
